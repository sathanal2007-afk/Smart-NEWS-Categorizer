# AI-Driven Smart News Agent & Categorizer

An intelligent, responsive web application that aggregates real-time global news headlines and dynamically categorizes them into custom segments like **Sports, Politics, Economy, Good News, and Bad News** based on user preference and advanced analysis.

---

## 🚀 Project Overview

The **AI News Agent** is a smart information aggregator designed to save users time by filtering content into explicit behavioral and subject-oriented buckets. Beyond traditional standard category filters (like Sports or Business), this project addresses user mindset by identifying the overall tone of information, splitting real-time content structurally into positive updates (**Good News**) or sensitive/critical reports (**Bad News**).

### 💡 Key Features
* **Real-Time Aggregation:** Integrates seamlessly with live News APIs to fetch current global headlines instantly.
* **Smart Categorization System:** Custom tags and query handling designed to slice news feeds dynamically into distinct segments.
* **Modern Adaptive UI:** Built using HTML5 and Tailwind CSS with full mobile responsiveness and smooth structural transitions.
* **Content Context Analysis Framework:** Designed to process textual elements and headers to match targeted categories accurately.

---

## 🛠️ Tech Stack & Architecture

* **Frontend Layout:** HTML5, Modern CSS via Tailwind CSS (CDN Integration)
* **Design Components:** FontAwesome Icons, Google Fonts System
* **Client-Side Logic:** Native Asynchronous JavaScript (Fetch API, Promises, ES6 DOM Manipulation Arrays)
* **Data Stream Engine:** JSON Web API Endpoint System (Live Data Synchronization)
